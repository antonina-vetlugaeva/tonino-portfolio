Как это выглядит

Задача

Создание веб-постера по теме проекта

Когда

Ноябрь 2018 – Декабрь 2018

Инструменты

Adobe Photoshop Adobe Indesign Atom Github

Создать новый жанр веб-искусства — задача не из легких. Но Б18ДЗО5 трудностей не боится!

Проблема

А может, просто сделать лендинг? Или же что-то, приближенное к плакату в привычном понимании? Об этом предстоит подумать. Главная задача — заинтересовать идеей проекта.

Решение

Комбинация рекламного текста, знакомой фирменной графики и интерактивных дополнений. Огромного количества информации этот плакат не даст, но заинтригует точно.

Поиск идей

Референсами к работе стали постеры, выполненные для задания по креативному проектированию. Я позаимствовала мотив исчезновения, а одно из анимированных изображений органично вписался в веб-плакат.

Элементы стиля

Адаптивная верстка

Адаптивная верстка меняет дизайн страницы в зависимости от платформы и размера экрана. Она позволяет существенно экономить и не отрисовывать новый дизайн для каждого разрешения, а менять размеры и расположение отдельных элементов.

CSS эффекты

Я использовала некоторые эффекты для воплощения метафоры исчезновения. Бледнеющие кнопки, исчезающие нежелательные слова в написанном мною четверостишие, пульсирующая тьма, почти поглощенная светом - спасибо коду за это (рифма случайна)
