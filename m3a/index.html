А можно плакатик, чтоб интересно было и с джаваскриптом? Попробуем. Простор для творчества огромен.

Проблема

Внедрение языка программирования javascript в веб-постер. Можно доработать плакат из второго модуля, а можно создать что-то совершенно новое и оригинальное.

Решение

Я пошла по второму пути, и превратила веб-постер в своеобразную игру, призванную якобы удалить любые введенные пользователем воспоминания.

Поиск идей

В проекте воплощена система kanban, знакомая всем по сервисам notion или trello. Однако, если названные сайты используются для запоминания какой-либо нужной информации, мой, напротив, помогает забыть ненужную.

Как это работает

Расслабьтесь. Вспомните что-то плохое. Запишите это. Пара движений — и воспоминание будет удалено! Не по-настоящему, конечно, но возможно, этот процесс вас хоть немного успокоит.

Резиновая верстка

Я использовала некоторые эффекты для воплощения метафоры исчезновения. Бледнеющие кнопки, исчезающие нежелательные слова в написанном мною четверостишие, пульсирующая тьма, почти поглощенная светом - спасибо коду за это (рифма случайна)

Javascript

Одной из ключевых задач JavaScript является взаимодействие с пользователем и манипуляция элементами веб-страницы. Добавлены лоадер, анимация набора текста, всплывающее уведомление и сама возможность переноса блоков с воспоминаниями из столбца в столбец.
